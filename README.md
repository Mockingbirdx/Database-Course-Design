# 数据库课设前端项目工程

## 环境依赖
```
node.js

vscode

git
```

## 配置环境
```
npm install

npm install element-plus --save

npm install typescript

vscode 推荐插件:
1. Vue
2. Vue Language Features (Volar)
```

### Compiles and hot-reloads for development

npm run serve
